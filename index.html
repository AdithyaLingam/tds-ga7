Reproducible Metrics (Code)


import pandas as pd
from pathlib import Path

# Load quarterly results
q = pd.read_csv(Path('data/quarterly.csv'))

# Compute YoY and QoQ deltas
q['rev_qoq'] = q['revenue'].pct_change()
q['rev_yoy'] = q['revenue'].pct_change(4)

# Screen for beats vs. consensus
beats = q[q['eps_actual'] > q['eps_consensus']]
print('Beat quarters:', beats['quarter'].tolist())

Financial Formulas

Net Present Value (discrete cash flows):

\[ \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} - I_0 \]

Dividend Discount (Gordon Growth):

\[ P_0 = \frac{D_1}{r - g} \quad \text{for } r > g \]
